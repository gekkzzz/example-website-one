# 🍏 Example Website One

A fully responsive, high-performance web application built to demonstrate modern frontend architecture, clean code practices, and seamless deployment workflows. 

Inspired by Apple's design philosophy, this project emphasizes a sleek, minimalist user interface with generous use of whitespace, crisp typography, and smooth, recognizable aesthetics that convey a premium brand feel.

**🔗 [View Live Site](https://example-website-one-coral.vercel.app/)**

---

## 🚀 Key Features

* **Apple-Inspired UI/UX:** A minimalist, clean design language focusing on content legibility, premium aesthetics, and smooth user interactions.
* **Responsive & Mobile-First:** Fluid layouts that ensure the site looks flawless on everything from smartphones to 4K monitors.
* **Modern Tooling:** Built with **TypeScript** and **Vite** for a blazing-fast development experience and highly optimized production builds.
* **Optimized Performance:** Achieves high Core Web Vitals scores through efficient asset loading, clean DOM structure, and modern bundling.
* **Continuous Integration:** Automated CI/CD pipeline via **Vercel** for instant, reliable deployments.

---

## 🛠️ Tech Stack

* **Language:** TypeScript, HTML5
* **Styling:** Vanilla CSS (Modern CSS3 features)
* **Build Tool:** Vite
* **Deployment:** Vercel

---

## 🧠 What I Learned

Building this project allowed me to dive deeper into:

1. **Strict Typing with TypeScript:** Catching errors at compile-time and writing more predictable, self-documenting code.
2. **Modern Build Tools:** Configuring and leveraging Vite for rapid Hot Module Replacement (HMR) and optimized build outputs.
3. **Advanced CSS Architecture:** Organizing styles for maintainability while recreating complex, premium UI design patterns (like Apple's signature layouts).
4. **Semantic HTML & Accessibility:** Improving screen-reader accessibility (A11y) and overall SEO through proper document structure.

## ⚙️ Local Development

To run this project locally, make sure you have [Node.js](https://nodejs.org/) installed on your machine.

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/gekkzzz/example-website-one.git](https://github.com/gekkzzz/example-website-one.git)
   ```

2. **Navigate to the project directory:**
   ```bash
   cd example-website-one
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Launch the local development server:**
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to the `localhost` URL provided in your terminal.
```
